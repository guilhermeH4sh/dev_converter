# Descrição

O script manipula um formulário de conversão de moedas. Quando o usuário envia o formulário, ele valida o valor digitado e a moeda selecionada. Se estiver tudo correto, o código calcula a conversão de reais para euro ou dólar usando taxas fixas. O resultado é exibido formatado com toLocaleString, acrescentando ícones, e uma pequena animação é aplicada ao valor convertido. Após a conversão, os campos do formulário são limpos.
